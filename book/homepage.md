# คู่มือการเตรียมความพร้อมเพื่อเข้าสู่ยุคควอนตัมสำหรับองค์กรไทย
**Post-Quantum Readiness Handbook for Thai Organizations**

---

ในอดีตคอมพิวเตอร์ควอนตัม (Quantum Computer) เคยเป็นเพียงทฤษฎีและการทดลองในห้องปฏิบัติการ แต่ปัจจุบันหลายประเทศได้มีการพัฒนาเทคโนโลยีนี้อย่างจริงจัง และมีความเป็นไปได้ที่จะมีการนำมาใช้งานเชิงพาณิชย์ในอีกไม่กี่ปีข้างหน้า สิ่งที่น่ากังวลคือคอมพิวเตอร์ควอนตัมมีความสามารถในการถอดรหัสลับข้อมูลที่สูงกว่าคอมพิวเตอร์ในยุคปัจจุบันอย่างมาก ทำให้อัลกอริทึมการเข้ารหัสลับที่ใช้อยู่ในปัจจุบันมีความเสี่ยงและอาจไม่สามารถใช้ปกป้องข้อมูลได้อีกต่อไป

คู่มือฉบับนี้จัดทำโดย **บริษัท ควอนตัม เทคโนโลยี ฟาวเดชั่น (ประเทศไทย) จำกัด** เพื่อนำเสนอข้อมูลและแนวทางในการเตรียมความพร้อม เปลี่ยนผ่านระบบการเข้ารหัสลับขององค์กรไทยสู่เทคโนโลยีที่ทนทานต่อภัยคุกคามจากคอมพิวเตอร์ควอนตัม โดยเนื้อหาส่วนหนึ่งในคู่มือฉบับนี้ถูกเผยแพร่ใน [คำแนะนำเรื่องแนวทางการปฏิบัติการเตรียมความพร้อมสำหรับยุคควอนตัม Guidelines for Post–Quantum Readiness](https://www.ncsa.or.th/ncsa_book/5004a6893034327c56000029) โดยศูนย์วิจัยความมั่นคงปลอดภัยไซเบอร์ สำนักงานคณะกรรมการการรักษาความมั่นคงปลอดภัยไซเบอร์แห่งชาติ (สกมช.)

```{warning}
รัฐบาลสหรัฐอเมริกาได้กำหนดเส้นตายการเปลี่ยนผ่านไปสู่ยุคควอนตัม (Quantum Migration Deadline) ภายในปี **ค.ศ. 2035 (พ.ศ. 2578)** ให้หน่วยงานภาครัฐและโครงสร้างพื้นฐานสำคัญ (critical infrastructure) จากนั้นรัฐบาลหลายประเทศทั่วโลกรวมถึงประเทศไทยต่างใช้กรอบเวลานี้กำหนดเส้นตายที่สอดคล้องกัน
```

```{danger}
ภัยคุกคามจากคอมพิวเตอร์ควอนตัมไม่ใช่เรื่องในอนาคตไกล — การโจมตีประเภท **Harvest Now, Decrypt Later** ซึ่งผู้ไม่หวังดีเก็บรวบรวมข้อมูลที่เข้ารหัสลับไว้ขณะนี้เพื่อถอดรหัสด้วยคอมพิวเตอร์ควอนตัมในอนาคต **กำลังเกิดขึ้นแล้วในปัจจุบัน** องค์กรที่ต้องรักษาความลับของข้อมูลในระยะยาวควรเริ่มวางแผนรับมือตั้งแต่วันนี้
```

## ผู้อ่านกลุ่มเป้าหมายของคู่มือฉบับนี้

คู่มือฉบับนี้เหมาะสำหรับผู้อ่านกลุ่มต่าง ๆ ดังนี้

- **ผู้บริหารระดับสูง (Executives)**: เพื่อทำความเข้าใจความเสี่ยงของเทคโนโลยีควอนตัม ทำให้สามารถกำหนดนโยบายองค์กรและลงทุนในโครงการใหม่ ๆ ในองค์กรเพื่อการบริหารจัดการความเสี่ยงได้อย่างมีประสิทธิภาพ

- **ผู้จัดการด้าน IT และความมั่นคงปลอดภัยสารสนเทศ (IT and Security Managers)**: วางแผนและบริหารจัดการการเปลี่ยนผ่านไปสู่ยุคควอนตัม (Quantum Migration) ขององค์กร ตั้งแต่การจัดทำ Roadmap การประเมินความเสี่ยง ไปจนถึงการประเมินงบประมาณที่จำเป็นต้องใช้

- **วิศวกรและนักพัฒนาระบบ (Software Engineers and Developers)**: เข้าใจข้อพิจารณาทางเทคนิคในการนำ PQC และ QKD ไปใช้งานจริง ครอบคลุมการเลือกอัลกอริทึม ข้อจำกัดของฮาร์ดแวร์และซอฟต์แวร์ เครื่องมือ Open Source รวมถึงกระบวนการรับรองมาตรฐาน

- **เจ้าหน้าที่ระดับปฏิบัติการและผู้ตรวจสอบ (Operators and Auditors)**: ใช้ Checklist รายการสินทรัพย์ทางสารสนเทศ (Crypto Assets Inventory) ทะเบียนความเสี่ยง (Risk Register) และแนวปฏิบัติเรื่อง Crypto Agility เพื่อดำเนินงานและตรวจสอบสถานะความพร้อมขององค์กรได้อย่างเป็นระบบ
